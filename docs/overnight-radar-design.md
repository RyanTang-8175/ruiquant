# 尾盘隔夜雷达（一夜持股法）设计 0.1

## 定位

尾盘隔夜雷达是一个独立短线模式，不纳入通用选股总分。它用于在尾盘筛选适合隔夜观察的股票，目标是捕捉次日冲高溢价，同时优先规避尾盘诱多和量化收割风险。

## 使用周期

- 筛选时间：当日尾盘
- 持仓周期：当日尾盘买入，次日处理
- 目标收益：次日冲高约 2%
- 核心纪律：先定义离场条件，再决定是否参与

## 时间规则

- 14:30-14:45：初筛，只生成候选，不买入
- 14:45-14:50：观察尾盘承接
- 14:50-14:55：最终确认
- 14:55 后：不追高，只允许放弃

## 初筛硬条件

### 涨幅

- 理想区间：3%-5%
- 可接受区间：2.5%-5.5%，必须标注偏弱或偏热
- 低于 2.5%：强度不足
- 高于 5.5%：偏热
- 高于 6%：追高/诱多风险

### 量比

- 严格要求：量比 >= 1.2
- 量比 < 1.2：排除

### 换手率

- 严格要求：5%-10%
- 低于 5%：排除，活跃度不足
- 高于 10%：排除，分歧或波动过大

### 流通市值

- 严格要求：50 亿-200 亿
- 低于 50 亿：排除，波动和操控风险偏高
- 高于 200 亿：排除，短线推动难度偏高
- 数据缺失：排除，并标注缺少流通市值数据
- 第一阶段数据源：AKShare
- 后续数据源：同花顺 iFinD API
- 策略层只读取统一字段，不直接依赖具体数据源

## K 线形态

### 硬条件

- MA5 > MA10 > MA20
- 当前价 > MA5
- 当前价处于当日振幅上方 70% 以上
- 上影线占当日振幅 <= 35%

### 近 5 日涨幅分级

- <= 10%：健康
- 10%-15%：可接受，偏热
- 15%-25%：高位风险，不建议隔夜
- > 25%：排除，接力风险过高

### 压力位过滤

- 当前价上方 3% 内存在近 20 日前高：标注压力风险
- 靠近前高且放量滞涨：排除

## 分时确认

### 硬条件

- 全天至少 70% 时间在分时均价线上方
- 14:30 后持续在分时均价线上方
- 14:50 前不能有效跌破分时均价线

### 有效跌破定义

价格低于分时均价线超过 0.3%，且持续 3 分钟以上。

### 尾盘确认

- 14:30-14:50 之间创出日内新高
- 创新高后允许回踩
- 回踩幅度 <= 1%
- 回踩后 3 分钟内不继续创新低
- 回踩后仍在分时均价线上方

## 诱多风险

以下情况标注高风险或排除：

- 14:45 后直线拉升超过 2%，但没有回踩
- 尾盘放量但价格不再创新高
- 14:50 后快速跳水
- 创高失败后跌破分时均价线
- 长上影、放量滞涨、连续大涨后的高换手

## 次日离场规则

- 次日浮盈 >= 2%：触发止盈提示
- 次日跌破当日分时均价线：触发止损提示
- 次日低开 <= -2%：标记高风险，观察 5-10 分钟；若不能收回分时均价线则离场
- 次日高开 >= 3%：不追高、不加仓，优先分批止盈
- 高开后快速跌破分时均价线：剩余仓位按计划离场

## 今日适用度

该方法必须先判断今日是否适用，再展示候选股。

### 高

- 大盘尾盘稳定或走强
- 主线板块活跃
- 候选股数量充足
- 跌停和大面股没有明显扩散
- 市场成交额正常或放大

### 中

- 大盘震荡
- 有局部题材机会
- 候选数量一般
- 风险可控，但需要精选

### 低

- 大盘尾盘走弱
- 题材轮动快
- 候选股少
- 炸板、冲高回落较多
- 次日溢价不确定

### 不适用

- 大盘尾盘跳水
- 高标退潮明显
- 跌停数量增加
- 主线板块集体走弱
- 重大节假日前或重大不确定事件前

## 输出状态

候选股不直接标记为“买入”，而使用以下状态：

- 初筛候选
- 等待确认
- 可执行
- 尾盘诱多风险
- 排除

每只股票展示：

- 匹配度
- 符合条件数量
- 未满足条件
- 主要风险
- 次日计划
- 数据更新时间

## 边界

该方法只是短线辅助筛选和纪律提醒，不构成投资建议。方法输出必须保留规则明细，不能只给一个黑箱分数。
