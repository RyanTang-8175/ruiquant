## 项目定位

**AlphaEye 不是聊天机器人，不是荐股工具，而是一个 iFinD 驱动的个人 A 股投研工作台。**

完整工作链路：
```
发现信息 → 判断真假热点 → 生成研究候选 → 公司深度研究
→ 证据化 AI 分析 → 写入研究审计 → T+1/T+2/T+3 回放
→ 比较评分命中率 → 累积研究记忆 → 策略评估
```

### 核心价值观

1. **数值程序算、AI 只解释、不编数据** — 所有分数和指标由程序本地计算，AI 只负责解释
2. **不推荐买卖，只做研究辅助** — 默认只给观察/模拟/验证/放弃，不输出买入卖出指令
3. **数据先行** — 没有可靠数据不做评分和预测；数据不够时降置信度但永远不编造
4. **预测必回填** — 每次研究假设自动进入审计，T+1/T+2/T+3 自动验证
5. **反量化风控** — 先看风险再看机会，尾盘诱多/高位接盘/分时脉冲/放量滞涨/板块背离五维扫描

## 数据来源全景图

### 行情数据流

```
用户请求 → get_realtime_quote()
  ├── 优先: iFinD real_time_quotation (20s TTL)
  │   └── 失败 → 腾讯 qt.gtimg.cn (30s TTL)
  │       └── 失败 → 新浪 hq.sinajs.cn (30s TTL)
  └── 返回: {price, change_pct, turnover, amount, volume_ratio, PE...}
```

### K线数据流

```
get_kline(code, period)
  ├── 优先: iFinD cmd_history_quotation (今日TTL=5min / 历史TTL=6h)
  │   └── 失败 → 东财 push2his.eastmoney.com
  │       └── 失败 → 新浪 money.finance.sina.com.cn
  │           └── 失败 → 腾讯 web.ifzq.gtimg.cn
  └── 分时图: period="1" → 1分钟K线 (240根)
```

### iFinD 额度管理 (核心红线)

| 端点 | 额度 (月) / 缓存 TTL | 用途 |
|---|---|---|
| real_time_quotation | 高频 / 20s | 实时行情 |
| smart_stock_picking | **4000次/月 / 60min** | 智能选股(最稀缺) |
| cmd_history_quotation | 中频 / 5min(今日)6h(历史) | K线数据 |
| report_query | 低频 / 6h | 公告/研报 |
| basic_data | 低频 / 6h | PE/ROE/总股本 |

### iFinD ≠ 基础数据，它是专业升级

- **基础数据**：腾讯/新浪免费行情 — 永久可用
- **专业升级**：iFinD — 更高精度、智能选股、公司深度研究
- **免费数据只是安全气囊，不是引擎。iFinD 是整个投研工作台的专业底盘。**


## 评分体系详解

### 六维评分框架

```
总评分 = Σ(维度分 × 权重) - 反量化惩罚

维度              权重    来源
热度 (heat)       30%    涨幅/量比/成交额/换手/振幅
承接 (support)    25%    均价线上方比例/收盘位置/上影线
题材 (theme)      15%    板块涨幅/涨停数/龙头/行业
延续 (continuation) 15%  均线/近5日涨幅/量价/压力空间
策略匹配 (strategy_match) 15% 隔夜/短持/回调/突破
反量化 (anti_quant) 惩罚  尾盘诱多/高位接盘/分时脉冲/放量滞涨/板块背离
```

### 五档研究评级 (2026-06-16新增)

| 条件 | 评级 | 含义 |
|---|---|---|
| 反量化"极高" 或 risk≥71 | 回避 | 风险不可接受 |
| 机会分≥72 且 风险≤35 | 强力研究 | 最高优先 |
| 机会分≥65 且 风险≤45 | 偏重研究 | 重点方向 |
| 机会分<45 或 风险>65 | 降低关注 | 暂缓 |
| 其他 | 持有观察 | 默认 |

### 扩展因子 (2026-06-16新增)

| 因子 | 说明 |
|---|---|
| 资金流 | 量价协同+背离+成交额强度+换手率+K线趋势 → 主力积极~出货五档 |
| 市场Regime | ADX+波动率 → 趋势上涨/下跌/高波/低波/震荡 |
| 估值偏离度 | 20/60日均线偏离+PE/PB → 显著低估~高估五档 |
