# 项目概览

## AlphaEye 是什么

**AlphaEye 是一个 iFinD 驱动的个人 A 股投研工作台**——不是聊天机器人，不是荐股工具。

完整工作链路：
```
发现信息 → 判断真假热点 → 生成研究候选 → 公司深度研究
→ 证据化 AI 分析 → 写入研究审计 → T+1/T+2/T+3 回放
→ 比较评分命中率 → 累积研究记忆 → 策略评估
```

## 核心价值观（不可违背的设计原则）

1. **数值程序算、AI 只解释、不编数据**——所有分数/指标由程序本地计算，AI 只负责解释
2. **不推荐买卖，只做研究辅助**——默认只给"观察/模拟/验证/放弃"，不输出买入卖出指令
3. **数据先行**——没有可靠数据不做评分和预测；数据不够时降置信度但永远不编造
4. **预测必回填**——每次研究假设自动进入审计，T+1/T+2/T+3 自动验证
5. **反量化风控**——先看风险再看机会，尾盘诱多/高位接盘/分时脉冲/放量滞涨/板块背离五维扫描
6. **iFinD 额度保护**——智能选股 4000次/月最稀缺，60分钟缓存保护，多Agent只读上下文不各自调API
7. **免费行情永久可用**——腾讯/新浪作为永久数据底盘，AI 永远不能说"数据暂缺"
8. **在现有基础上添加，不推翻重来**——所有改动都是增量，不改变现有架构和风格

## 技术栈

- 前端：Streamlit（Mobile First，max-width 1120px，白色背景）
- 后端：Python 3.10+
- 数据库：SQLite（评分/审计/对话记忆）
- 主模型：DeepSeek v3/v4（OpenAI 兼容接口）
- 备选模型：Mimo（小米，自动切换）
- 专业数据：iFinD（同花顺）——行情/智能选股/公司研究/公告/基本面
- 免费数据：腾讯/新浪/东财——行情兜底/K线/新闻
- 定时任务：APScheduler

## 项目规模

- 约 11,040 行 Python，100 个 .py 文件
- 7 个主导航页面：行情/雷达/研究/AI/审计/交易/我的
- +2 个子页面：个股详情/登录
- 98 个自动化测试，全部通过

## 核心数据流

```
实时行情: iFinD (20s TTL) → 腾讯 (30s) → 新浪 (30s)
K线数据:  iFinD (今日5min/历史6h) → 东财 → 新浪 → 腾讯
智能选股: iFinD smart_stock_picking (4000次/月, 60min TTL)
新闻/公告: 新浪/东财/财联社/腾讯/互动易/巨潮
```

# 已实现的全部功能（不要再建议这些）

## 评分体系
- **六维评分**：热度(30%)+承接(25%)+题材(15%)+延续(15%)+策略匹配(15%)，反量化惩罚
- **五档研究评级**：强力研究/偏重研究/持有观察/降低关注/回避（基于机会分×风险分交叉判断）
- **扩展因子**：资金流(量价协同+背离)、市场Regime(ADX+波动率)、估值偏离度(均线偏离+PE/PB)
- **因子正交化**：新因子R²>0.7时提示冗余
- 评分区间已拉宽（热度权重20%→30%，涨幅区间10~85→5~95）

## AI 能力
- **多Agent并行分析**：技术面/基本面/资金面/风控 四个Agent同时跑（ThreadPoolExecutor）
- **多空辩论**：bull🐂 vs bear🐻 对抗校验，两个Agent并行辩论后以折叠卡片展示
- **10个投资提示词模板**：宏观传导/财报拆解/技术扫描/估值分析/空头压测等9个新场景
- **实时行情护栏**：每轮对话注入最高优先级system消息，护栏>工具缓存
- **兜底永远带真实数据**：DeepSeek挂了也用腾讯免费行情出K线/支撑位/反量化
- **iFinD研报引用**：AI输出带 `_citation_sources`，标注[公告][研报]
- **Mimo备选模型**：DeepSeek失败自动切换
- **`_extract_stock_code`修复**：只从用户原文提取代码，防用户画像串标

## 图表与可视化
- **K线多副图**：K线主图+成交量柱+MACD(DIF/DEA/柱)+RSI(70/30参考线) 四副图 Plotly
- **分时图**：1分钟K线双副图（价格线+昨收参考线+成交量柱）
- **收益走势图**：交易页基于已卖出交易的累计盈亏曲线
- **资金曲线模拟**：凯利仓位计算器 Plotly 资金曲线
- 全站颜色统一为 `var(--red)`/`var(--green)` CSS 变量

## 策略与审计
- **凯利仓位计算器**：胜率×盈亏比×止损→仓位百分比+资金曲线
- **策略标准化评估**：夏普/最大回撤/VaR(95%)/Sortino/胜率一键计算
- **策略下线规则**：回撤>15%暂停/胜率<30%降仓/夏普<0下线（4条规则，监控面板展示）
- **T+1/T+2/T+3 审计回填**（命中率计算已修复，分母从backfill行数改为VerificationRecord数）
- **归因画像**：策略效果/AI准确率/用户执行纪律三类归因
- **评分对比**：iFinD新评分 vs 旧六维评分命中率对比

## 雷达与选股
- **候选池三维度交叉**：涨幅+成交额+换手率去重（修复了"日日推同一批"的bug）
- **自然语言→iFinD**：输入"新能源车""高股息"自动映射为iFinD查询词
- **信息雷达**：免费源(东财人气榜)+iFinD精准层+伪热点过滤
- **产业链Chokepoint SOP**：6阶段拆解（产业链映射→Chokepoint识别→标的筛选→硬数据验证→研究清单→人工复核）

## 系统与安全
- 22处裸except→Exception，2处DB连接泄漏修复
- 3个全局缓存加 threading.Lock
- Playbook JSON损坏保护
- st.rerun() DOM竞态修复（全部改为 _nav_pending 标志）
- 量化系统监控面板：数据延迟/iFinD额度/AI服务/审计健康
- 持仓追踪升级：资产总览5列+收益曲线+持仓市值占比
