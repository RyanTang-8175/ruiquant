# 关键决策与待办事项

## 已确认的关键决策

### 1. 技术选型

| 层 | 选择 | 原因 |
|----|------|------|
| 数据源 | baostock | VPN 环境下 AKShare 无法使用 |
| AI 后端 | DeepSeek | 便宜（¥2/百万token）、中国能用、中文好 |
| 前端 | Streamlit | 用户不会前端，纯 Python |
| 数据库 | SQLite | 轻量，个人使用够用 |
| 部署 | 阿里云 VPS | 需要公网访问 |

### 2. 产品定位

- **不是**自动交易机器人
- **不是**万能量化框架
- **是**个人 A 股研究 + 模拟训练 + AI 对话助手

### 3. 核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| 市场概览 | 一眼看市场全貌 | ✅ 已完成 |
| 选股/观察池 | 35 因子评分 | ✅ 已完成 |
| AI 对话 | DeepSeek 集成 | ✅ 已完成（需配置 API Key） |
| 模拟盘 | 真实规则模拟交易 | ✅ 已完成 |
| AI 预测 | 预测 + 回填 | ✅ 已完成 |
| AI 教练 | 交易评价 + 建议 | ✅ 已完成 |
| 自我学习 | 因子权重自动调整 | ✅ 已完成 |
| 股票详情 | K 线图 + 技术指标 | ✅ 已完成 |

### 4. 评分引擎

- **35 个因子**，5 大类
- **IC 加权**：基于学术研究的 IC 值
- **自动优化**：每月根据准确率调整权重
- **评级**：强关注(80+) / 观察(65-79) / 中性(50-64) / 不追(0-49)

### 5. 模拟盘规则

- **T+1**：当天买入不能当天卖出
- **涨跌停**：涨停不能买，跌停不能卖
- **费用**：佣金万2.5、印花税千0.5、过户费万0.2
- **仓位**：单票 30%，总仓位 80%
- **止损**：8% 提醒，15% 强制卖出
